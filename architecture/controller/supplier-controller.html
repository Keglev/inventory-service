POST /api/suppliers

Create new supplier

Request

POST /api/suppliers
Authorization: Bearer <token>
Content-Type: application/json

{
  "name": "New Supplier Inc",
  "contactName": "Manager Name",
  "phone": "555-9999",
  "email": "manager@newsupplier.com",
  "address": "789 Pine St, Chicago, IL"
}

Authorization

@PreAuthorize("hasRole('ADMIN') and !@securityService.isDemo()")

  • ‚úÖ ADMIN users (not in demo mode)
  • ‚ùå USER users (all blocked)
  • ‚ùå Demo users (all blocked, even ADMIN)

Request Body

@NotNull(message = "Name must not be null")
@NotBlank(message = "Name must not be blank")
String name;

@NotNull(message = "Contact name must not be null")
String contactName;

String phone;      // Optional
String email;      // Optional
String address;    // Optional

Validation Rules

  • name: Required, non-blank
  • contactName: Required, non-null
  • id: Must be null (server generates ID)
