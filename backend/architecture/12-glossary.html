Glossary

Domain, technical, and infrastructure terms used across this documentation, defined in the context of SmartSupplyPro. Cross-references point to the section or decision where a term is treated in depth.

Term Definition
ADB (Oracle Autonomous Database) Managed Oracle 23ai database backing the prod profile; reached via the wallet. See ADR-0001
ADR Architecture Decision Record — a decision with its justification and rejected alternatives. See §9
AnalyticsService Service computing stock valuation and trends, including WAC and FIFO analysis
AppUser JPA entity for an authenticated user; carries the assigned role
arc42 The documentation template structuring this architecture (§1–§12)
BusinessExceptionHandler @ControllerAdvice mapping domain exceptions to HTTP status: InvalidRequest → 400, Duplicate / IllegalState → 409
cwallet.sso Oracle auto-login wallet file; opens the wallet for mTLS without a wallet password at runtime. See ADR-0001
DTO (Data Transfer Object) The only object types that cross the controller boundary; 14 in the dto/ package. See ADR-0003
@EntityGraph JPA fetch hint on InventoryItemRepository loading supplier in a single join to avoid N+1 queries
ErrorResponse The canonical three-field error record {error, message, timestamp}; error = HttpStatus.name().toLowerCase(). See ADR-0004
FIFO First-In-First-Out inventory costing method used by AnalyticsService
Fly.io Hosting platform for the backend (port 8081, health check /api/health)
GlobalExceptionHandler @ControllerAdvice for framework and uncaught exceptions; applies sanitize(). See ADR-0005
HikariCP JDBC connection pool, sized for Fly.io memory constraints
InventoryItem Core domain entity representing a tracked stock item, linked to a Supplier
JaCoCo Java code-coverage tool run in the CI pipeline
Koyeb Hosting platform for the React frontend; the cross-origin request origin
mTLS Mutual TLS — certificate-based transport authentication provided by the Oracle wallet. See ADR-0001
N+1 problem Repeated per-row association queries; avoided here via @EntityGraph
OAuth2 Authentication via Google; on success the flow redirects to /auth. See §6
@PrePersist / @CreationTimestamp JPA / Hibernate callbacks that populate createdAt (and default createdBy) before insert
RBAC Role-based access control; ADMIN and USER roles enforced via @PreAuthorize
sanitize() GlobalExceptionHandler method stripping file paths, class names, SQL, and credentials from error messages. See ADR-0005
SameSite=None; Secure Session cookie attributes enabling the cross-origin Koyeb → Fly.io session. See §6
StockChangeReason Enum classifying each stock movement (INITIAL_STOCK, MANUAL_UPDATE, SOLD, …). See the enums reference
StockHistory Domain entity recording each stock movement as an audit trail
Supplier Domain entity representing a goods supplier linked to inventory items
TNS_ADMIN Environment variable pointing the JDBC driver at the wallet directory
Trivy Container-image CVE scanner run in the Docker build workflow
WAC Weighted Average Cost inventory costing method used by AnalyticsService
Wallet (Oracle) Directory of TLS certificates and config (cwallet.sso, tnsnames.ora, …) authenticating the DB connection. See ADR-0001
