frontend / api/analytics/suppliers / getSuppliersLite

Function: getSuppliersLite()

getSuppliersLite(): Promise<SupplierRef[]>

Defined in: api/analytics/suppliers.ts:23

Returns a lightweight {id, name} list for populating supplier dropdowns. Silently returns an empty array on network or parse errors so filter controls degrade gracefully rather than blocking the UI.

Calls GET /api/suppliers?limit=200.

Returns

Promise<SupplierRef[]>

Example

const suppliers = await getSuppliersLite();
return <Select options={suppliers} />;
