Function: useItemDetailsQuery()

useItemDetailsQuery(itemId): UseQueryResult<ItemDetails | null, Error>

Defined in: api/inventory/hooks/useItemDetailsQuery.ts:36

Fetches full details for a single inventory item by ID.

Disabled until itemId is truthy — prevents a fetch with a null/undefined key. 30 s stale time keeps data fresh during a typical edit session while avoiding redundant re-fetches if the user reopens the same item shortly after.

The backend (InventoryItemDTO) sends the stock quantity as quantity and the price as price.

Parameters

itemId

ID of the item to fetch; pass null or undefined when no item is selected

string | null | undefined

Returns

UseQueryResult<ItemDetails | null, Error>

React Query result with normalized ItemDetails, or null on error / no selection

Example

const { data: itemDetails } = useItemDetailsQuery(selectedItem?.id);
