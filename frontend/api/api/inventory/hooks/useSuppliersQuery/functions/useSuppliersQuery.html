Function: useSuppliersQuery()

useSuppliersQuery(enabled): UseQueryResult<SupplierOption[], Error>

Defined in: api/inventory/hooks/useSuppliersQuery.ts:31

Loads a lightweight supplier list for selector controls.

The enabled parameter lets callers defer the fetch until the control that needs suppliers is actually visible (e.g. when a dialog opens), avoiding an unnecessary network request on every render.

Backend name is mapped to label to satisfy the { id, label } contract expected by UI option-list components.

Parameters

enabled

boolean

Pass true when the supplier selector is mounted/open; false suppresses the request entirely.

Returns

UseQueryResult<SupplierOption[], Error>

React Query result whose data is a SupplierOption[].

Example

const { data: suppliers, isLoading } = useSuppliersQuery(dialogOpen);
