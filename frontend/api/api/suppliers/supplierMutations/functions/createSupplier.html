Function: createSupplier()

createSupplier(supplier): Promise<{ error?: string; success: SupplierRow | null; }>

Defined in: api/suppliers/supplierMutations.ts:35

Creates a new supplier via POST /api/suppliers.

Parameters

supplier

SupplierDTO

DTO to POST; omit id and createdAt — both are auto-filled by the backend.

Returns

Promise<{ error?: string; success: SupplierRow | null; }>

Created supplier with system-generated id, or error message

Example

const { success, error } = await createSupplier({
  name: 'ACME Corp',
  email: 'contact@acme.com'
});
if (success) console.log(success.id);
