Interface: ItemDetails

Defined in: src/api/inventory/types.ts:61

Full item details fetched from backend /api/inventory/{id} endpoint. Contains complete item information including actual current values. Used by forms and dialogs to display/edit full item data.

Remarks

This interface represents the complete item data fetched after user selects an item. Use this for displaying actual current values and pre-filling forms.
