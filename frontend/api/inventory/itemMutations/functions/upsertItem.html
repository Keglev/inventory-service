Function: upsertItem()

upsertItem(req): Promise<UpsertItemResponse>

Defined in: src/api/inventory/itemMutations.ts:51

Create or update an inventory item. ID absent in request ‚áí create (POST); ID present ‚áí update (PUT).

Parameters

Returns

Example

  name: 'Widget A',
  supplierId: 'SUP-001',
  quantity: 100,
  price: 25.50
});

// Update existing item

  id: 'ITEM-123',
  name: 'Widget A v2',
  supplierId: 'SUP-001',
  quantity: 50,
  price: 27.00
});
