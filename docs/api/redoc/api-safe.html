What to do if the interactive docs fail

  1. Open api.html in a modern browser. It uses ReDoc from a CDN.
  2. If the page errors due to CSP or network restrictions, download the OpenAPI YAML and view it locally with a ReDoc CLI or editor that supports OpenAPI.
  3. Contact the docs maintainers or open an issue if the generated ReDoc is repeatedly failing.
